# Jie Lin Resume Site

Technologies used to build and run this site:

- React 19 with Vite for the app shell and bundling
- Tailwind CSS for utility styling plus custom glassmorphism helpers
- motion for scroll and hover animations
- lucide-react for iconography
- PostCSS with autoprefixer and tailwindcss plugins for CSS processing
- ESLint for basic linting support
- npm scripts for local dev (`npm run dev`), production builds (`npm run build`), and previews (`npm run preview`)
